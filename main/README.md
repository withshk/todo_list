# 📝 C언어 To-Do List 프로젝트

CLI(Command Line Interface) 환경에서 동작하는 간단한 할 일 관리 프로그램입니다.
C언어의 구조체와 배열(또는 포인터)을 활용하여 기본적인 CRUD(생성, 조회, 수정, 삭제) 로직을 구현합니다.

## 🚀 주요 기능 (구현 예정)

- 새로운 할 일 추가 (Create)
- 전체 할 일 목록 조회 (Read)
- 할 일 완료 상태 변경 (Update)
- 특정 할 일 삭제 (Delete)

## 🛠️ 빌드 및 실행 방법

터미널(또는 명령 프롬프트)에서 아래 명령어를 입력하여 실행합니다.

```bash
gcc main.c -o todo
./todo
```
